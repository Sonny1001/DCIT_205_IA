Introduction to Web Development

Here are some of the topics you will come across when you enroll into our Web Development programme:

  1. HTML (Hypertext Markup Language)

    HTML is the backbone of any web page. It provides the structure and content of a webpage through a set of tags. These tags define elements such as headings, paragraphs, images, links, and more.

  2. CSS (Cascading Style Sheets)

    CSS is used to style and format HTML elements. It allows you to control the layout, colors, fonts, and overall visual presentation of your web pages. CSS can be applied inline, in a separate stylesheet, or within the HTML document.

  3. JavaScript

    JavaScript is a powerful scripting language that enables dynamic and interactive web pages. It runs in the browser and can be used to manipulate the content, handle user interactions, and make asynchronous requests to servers (AJA)).

  4. Front-End Development

    Front-end development focuses on the client side of web development, i.e., what users see and interact with. Modern front-end frameworks/libraries like React, Angular, and Vue.js make it easier to build complex user interfaces and manage state.

  5. Back-End Development

    Back-end development involves server-side programming. Common back-end languages include Node.js, Python, Ruby, PHP, and Java. Back-end developers handle server logic, databases, and server-client communication.

  6. Web Servers and Databases

    Understanding how web servers (e.g., Apache, Nginx) work and interacting with databases (e.g., MySQL, MongoDB) is crucial for building scalable and dynamic web applications.

  7. Version Control (Git)

    Version control is essential for tracking changes in your codebase, collaborating with others, and rolling back to previous versions if needed. Git is a widely used version control system.

  8. APIs (Application Programming Interfaces)

    APIs allow different software systems to communicate with each other. Web developers often work with APIs to integrate third-party services or build more complex applications.

  9. WebAssembly (Wasm)

    WebAssembly is a binary instruction format that enables high-performance execution of code on web browsers. It allows developers to run code written in languages like C, C++, and Rust at near-native speed, opening up new possibilities for performance-intensive web applications.

  10. Serverless Architecture

    Serverless computing allows developers to build and run applications without managing server infrastructure. Cloud providers (e.g., AWS Lambda, Azure Functions) handle the scaling and execution of code in response to events, making development more scalable and cost-effective.
